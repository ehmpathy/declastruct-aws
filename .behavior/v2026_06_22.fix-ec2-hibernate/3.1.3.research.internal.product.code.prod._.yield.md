## pattern.5 = bastion auto-start [REUSE]

automatically starts bastion instance when tunnel opens.

**location**: `src/domain.operations/vpcTunnel/setVpcTunnel.ts`

**key mechanism**:
- check session status before tunnel operations [15]
- set session to active if not already [16]

**citations**:
- [15] `src/domain.operations/vpcTunnel/setVpcTunnel.ts:118-121`
  > `const bastionSession = await getEc2InstanceSession( { by: { instance: input.via.bastion } }, context, );`
