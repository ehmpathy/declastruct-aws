# blueprint: AWS Organizations Service Control Policy support

## summary

add SCP (Service Control Policy) management to declastruct-aws:
- `DeclaredAwsOrganizationServiceControlPolicy` domain object
- `DeclaredAwsOrganizationServiceControlPolicyAttachment` domain object
- DAOs with get/set/del operations
- integration tests for all operations

### domain operations — SCP

```
getOneOrganizationServiceControlPolicy
├── [+] input: { by: PickOne<{ primary, unique, ref }> }
├── [+] byPrimary: call DescribePolicy API
├── [+] byUnique: call ListPolicies, filter by name
├── [+] byRef: dispatch to byPrimary or byUnique
├── [+] duplicate check: throw ConstraintError if multiple found
└── [←] reuse: getAwsOrganizationsClient from organization

getAllOrganizationServiceControlPolicies
├── [+] input: {} (no filter)
├── [+] call ListPolicies(Type=SERVICE_CONTROL_POLICY)
├── [+] paginate if needed
└── [+] return: HasReadonly<typeof DeclaredAwsOrganizationServiceControlPolicy>[]

setOrganizationServiceControlPolicy
├── [+] input: PickOne<{ findsert, upsert }>
├── [+] findsert: lookup by name, return if found, create if not
├── [+] upsert: lookup by name, update if found, create if not
├── [+] create: call CreatePolicy API
├── [+] update: call UpdatePolicy API
└── [+] idempotency key: use name as dedup

delOrganizationServiceControlPolicy
├── [+] input: Ref<typeof DeclaredAwsOrganizationServiceControlPolicy>
├── [+] lookup: getOne by ref
├── [+] validate: check not attached (list targets)
├── [+] delete: call DeletePolicy API
└── [+] idempotent: return { deleted: true } if not found

castIntoDeclaredAwsOrganizationServiceControlPolicy
├── [+] input: Policy from AWS SDK
├── [+] parse content JSON to DeclaredAwsIamPolicyDocument
└── [+] return: HasReadonly<typeof DeclaredAwsOrganizationServiceControlPolicy>
```

### domain operations — SCP attachment

```
getOneOrganizationServiceControlPolicyAttachment
├── [+] input: { by: { unique: { policy, target } } }
├── [+] lookup policy by name via getOneOrganizationServiceControlPolicy
├── [+] call ListTargetsForPolicy, filter by target id
└── [+] return: attachment or null

getAllOrganizationServiceControlPolicyAttachments
├── [+] input: PickOne<{ forPolicy, forTarget }>
├── [+] forPolicy: call ListTargetsForPolicy
├── [+] forTarget: call ListPoliciesForTarget(Type=SERVICE_CONTROL_POLICY)
└── [+] return: attachment[]

setOrganizationServiceControlPolicyAttachment
├── [+] input: PickOne<{ findsert }>
├── [+] findsert: lookup, return if attached, attach if not
├── [+] lookup policy: getOne by name (RefByUnique)
├── [+] validate target: ensure org/account exists
├── [+] attach: call AttachPolicy API
├── [+] handle DuplicatePolicyAttachmentException as success
└── [+] return: the attachment

delOrganizationServiceControlPolicyAttachment
├── [+] input: { policy: RefByUnique, target: Ref }
├── [+] lookup policy: getOne by name
├── [+] validate: prevent detach if last SCP (AWS enforces this)
├── [+] detach: call DetachPolicy API
├── [+] handle PolicyNotAttachedException as success
└── [+] return: { deleted: true }
```

## constraints

### AWS constraints

per 3.1.1.research.external.product.access._.yield.md:

| constraint | description | mitigation | research citation |
|------------|-------------|------------|-------------------|
| unique name | SCP names must be unique per org | detect duplicates, throw ConstraintError | access.yield [2] DuplicatePolicyException |
| 5KB limit | SCP content max 5120 bytes | validate before API call | access.yield [9] quotas |
| last SCP | cannot detach last SCP from target | AWS enforces; let error propagate | access.yield [4] MIN_POLICY_TYPE_ATTACHMENT_LIMIT_EXCEEDED |
| management account | operations require org management account | document credential requirement | access.yield [2] authorization |
| detach before delete | policy must have no targets to delete | list targets, block if attached | access.yield [5] PolicyInUseException |

per 3.1.1.research.external.product.claims._.yield.md:

| constraint | description | research citation |
|------------|-------------|-------------------|
| SCPs set upper bound | permissions = intersection of SCP + IAM | claims.yield [1][2][3][4] converge |
| API not explicitly idempotent | findsert is client-side pattern | claims.yield [SUMP] API operations |

### design constraints

| constraint | description |
|------------|-------------|
| idempotency | all set operations must be idempotent |
| findsert pattern | lookup first, return extant if found |
| error clarity | ConstraintError with hint for actionable errors |
| snapshot coverage | all contract outputs must have snapshots |
