# distill: experience reproductions for AWS Organizations SCP support

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| create SCP | `dao.set.findsert(scp, context)` | instantiate domain object, call findsert | SCP created in AWS, returned with id/arn | integration |
| attach to org root | `dao.set.findetert(attachment, context)` | instantiate attachment, call findsert | attachment created, org root protected | integration |
| attach to account | `dao.set.findsert(attachment, context)` | instantiate attachment, call findsert | attachment created, account protected | integration |
| get by name | `dao.get.one.byUnique({ name }, context)` | call get with unique key | SCP returned or null | integration |
| get by id | `dao.get.one.byPrimary({ id }, context)` | call get with primary key | SCP returned or null | integration |
| delete SCP | `dao.set.delete(ref, context)` | call delete with ref | SCP removed from AWS | integration |
| idempotent findsert | `dao.set.findsert(scp, context)` (twice) | call findsert when SCP exists | returns extant, no duplicate | integration |
| duplicate name error | `dao.set.findsert(scp, context)` | call findsert when duplicate names exist | ConstraintError thrown | integration |

## journey test sketches

### journey 1: create and attach SCP

#### step table

| step | action | user sees |
|------|--------|-----------|
| t0 | before any changes | no SCP with name 'deny-snapshot-share' exists |
| t1 | findsert SCP | SCP created with id 'p-xxx', arn populated |
| t2 | findsert attachment to org root | attachment created, SCP attached to org |
| t3 | findsert again (idempotent) | returns same SCP, no new creation |

#### input/output pairs

##### t0 before state (snapshot target)
```typescript
// input
const scpBefore = await daoScp.get.one.byUnique({ name: 'deny-snapshot-share' }, context);

// output
null
```

##### t1 create SCP (snapshot target)
```typescript
// input
const scp = await daoScp.set.findsert(
  new DeclaredAwsOrganizationServiceControlPolicy({
    name: 'deny-snapshot-share',
    description: 'prevent RDS/EBS snapshot exfiltration',
    content: new DeclaredAwsIamPolicyDocument({
      statements: [
        new DeclaredAwsIamPolicyStatement({
          sid: 'DenySnapshotShare',
          effect: 'Deny',
          action: ['rds:ModifyDBSnapshotAttribute', 'ec2:ModifySnapshotAttribute'],
          resource: ['*'],
        }),
      ],
    }),
    tags: null,
  }),
  context,
);

// output
{
  id: 'p-abc123',
  arn: 'arn:aws:organizations::123456789012:policy/o-xxx/service_control_policy/p-abc123',
  name: 'deny-snapshot-share',
  description: 'prevent RDS/EBS snapshot exfiltration',
  content: { statements: [...] },
  tags: null,
}
```

##### t2 attach to org root (snapshot target)
```typescript
// input
const attachment = await daoScpAttachment.set.findsert(
  new DeclaredAwsOrganizationServiceControlPolicyAttachment({
    policy: { name: 'deny-snapshot-share' },
    target: { id: organization.id },  // 'r-xxx'
  }),
  context,
);

// output
{
  policy: { name: 'deny-snapshot-share' },
  target: { id: 'r-xxxx' },
}
```

##### t3 idempotent findsert (snapshot target)
```typescript
// input
const scpAgain = await daoScp.set.findsert(
  new DeclaredAwsOrganizationServiceControlPolicy({
    name: 'deny-snapshot-share',
    ...  // same content
  }),
  context,
);

// output
{
  id: 'p-abc123',  // same id as t1
  ...
}
```

### journey 3: delete SCP (requires detach first)

#### step table

| step | action | user sees |
|------|--------|-----------|
| t0 | setup: create and attach SCP | SCP attached to org root |
| t1 | attempt delete while attached | PolicyInUseException (clear error) |
| t2 | detach from org root | attachment removed |
| t3 | delete SCP | SCP removed, returns { deleted: true } |
| t4 | get after delete | returns null |

#### input/output pairs

##### t1 delete while attached (error snapshot)
```typescript
// input
await daoScp.set.delete({ name: 'deny-snapshot-share' }, context);

// output (error)
ConstraintError: cannot delete SCP 'deny-snapshot-share': still attached to 1 target(s)
  hint: detach from all targets before delete
  targets: ['r-xxxx']
```

##### t3 delete after detach (snapshot target)
```typescript
// input
await daoScp.set.delete({ name: 'deny-snapshot-share' }, context);

// output
{ deleted: true }
```

## critical paths

| critical path | description | why critical |
|---------------|-------------|--------------|
| findsert SCP | create or find SCP by name | primary usecase; must be idempotent |
| attach to org root | attach SCP to organization root | applies guardrails org-wide |
| get by unique | lookup SCP by name | needed for verification and references |
| idempotent behavior | repeated findsert returns same result | CI/CD requires safe re-apply |

## reproduction feasibility

### available test utilities

- `getMockedAwsApiContext` — mock context for unit tests
- `getSampleAwsApiContext` — real credentials for integration tests
- `useBeforeAll` from test-fns — shared async setup
- `given`, `when`, `then` from test-fns — BDD structure

### setup required

- **integration tests**: org management account credentials
- **unit tests**: mock OrganizationsClient send method

### concrete test sketch

```typescript
// scp.play.integration.test.ts
describe('DeclaredAwsOrganizationServiceControlPolicy', () => {
  const context = useBeforeAll(() => getSampleAwsApiContext());

  given('[case1] findsert creates when not found', () => {
    const scpName = `test-scp-${getUuid()}`;  // unique per run

    when('[t0] before any changes', () => {
      then('SCP does not exist', async () => {
        const found = await daoScp.get.one.byUnique({ name: scpName }, context);
        expect(found).toBeNull();
      });
    });

    when('[t1] findsert is called', () => {
      const scp = useThen('SCP is created', async () =>
        daoScp.set.findsert(
          new DeclaredAwsOrganizationServiceControlPolicy({
            name: scpName,
            description: 'test policy',
            content: testPolicyDocument,
            tags: null,
          }),
          context,
        ),
      );

      then('returns SCP with id', () => {
        expect(scp.id).toMatch(/^p-/);
      });

      then('output matches snapshot', () => {
        expect(scp).toMatchSnapshot();
      });
    });

    when('[t2] findsert is called again', () => {
      const scpAgain = useThen('returns extant', async () =>
        daoScp.set.findsert(
          new DeclaredAwsOrganizationServiceControlPolicy({
            name: scpName,
            description: 'test policy',
            content: testPolicyDocument,
            tags: null,
          }),
          context,
        ),
      );

      then('returns same id (idempotent)', () => {
        expect(scpAgain.id).toEqual(scp.id);
      });
    });
  });
});
```

## gaps

### integration test credentials

**blocked**: integration tests require org management account credentials.

**required to unblock**:
- credential setup via keyrack or environment variable
- test org with management account access

**blocker or deferred**: blocker for integration tests; unit tests can proceed without credentials

### OU attachment

**blocked**: attachment to OU requires `DeclaredAwsOrganizationUnit` which doesn't exist.

**required to unblock**: implement `DeclaredAwsOrganizationUnit` domain object

**blocker or deferred**: deferred — org root + account covers primary usecases per vision

### duplicate name setup

**blocked**: to test duplicate name detection, need two SCPs with same name.

**required to unblock**:
- manual setup in test org, OR
- test via mock in unit test

**blocker or deferred**: can test via unit test mock; integration test optional
