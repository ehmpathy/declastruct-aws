# handoff: declastruct-aws account alias

## .what

request `DeclaredAwsIamAccountAlias` in declastruct-aws

## .why

account aliases provide:
- human-readable identifier in AWS console header
- safety: prevents accidental changes to wrong account
- friendly sign-in url: `https://ahbode-prep.signin.aws.amazon.com`

currently managed via terraform (`terraform/product/iam.account_alias.tf`), but should be declastruct for consistency with other auth resources.

## .api

AWS SDK:
- `CreateAccountAliasCommand` - create alias
- `DeleteAccountAliasCommand` - delete alias
- `ListAccountAliasesCommand` - list aliases (max 1 per account)

## .domain object

```ts
interface DeclaredAwsIamAccountAlias {
  /**
   * the account alias (3-63 chars, lowercase, hyphen allowed, must start with letter)
   * .unique = true (only one alias per account)
   */
  alias: string;
}

export class DeclaredAwsIamAccountAlias
  extends DomainEntity<DeclaredAwsIamAccountAlias>
  implements DeclaredAwsIamAccountAlias
{
  public static unique = ['alias'] as const;
}
```

## .operations

| operation | behavior |
|-----------|----------|
| get | `ListAccountAliasesCommand` → return first or null |
| set | if extant differs: delete old, create new |
| del | `DeleteAccountAliasCommand` |

## .usage

```ts
const accountAlias = new DeclaredAwsIamAccountAlias({
  alias: 'ahbode-prep',
});
```
