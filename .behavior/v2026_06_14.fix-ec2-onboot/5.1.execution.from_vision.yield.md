- [x] **`upsert` silent no-op** — now throws explicit error about immutability in both files
- [x] **`as` cast in setEc2Instance.ts** — fixed by importing `DeclaredAwsEc2LaunchTemplate` as value (not type) for `isRefByPrimary`
- [x] **`as` cast in castIntoDeclaredAwsEc2Instance.ts** — replaced with explicit IIFE mapping AWS state names
- [x] **`tags?` undefined attribute** — changed to `tags: Record<string, string> | null` in `DeclaredAwsEc2LaunchTemplate`
- [x] **`state` vs `status` field name** — fixed in acceptance test template comment
- [x] **transitional state mapping** — `pending` now maps to `active`, `stopping`/`shutting-down` map to `stopped` in `getEc2InstanceSession`

- [x] **tags cast returns null** — `castIntoDeclaredAwsEc2LaunchTemplate` now returns `null` for empty tags
